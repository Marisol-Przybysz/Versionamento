Versionamento

O versionamento é necessário para que haja um controle das versões do arquivo html, previnindo a perca da versão do arquivo em caso de diversas modificações e organizando de modo que seja fácil para o desenvolvedor controlar as versões do arquivo.

Programas e plataformas

  1. GitHub
  2. GitHubDesktop
  3. Git

Comandos

  1. Commit
  2. Publish
  3. Push
  4. Fetch
  5. Merge
